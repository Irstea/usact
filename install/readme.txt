News
===========
Version 0.4 du 26 octobre 2015
--------------
Correction d'anomalies :
- ajout de la liste des entretiens dans les interventions
- la table position_usage_activite est maintenant modifiable
- il est possible de créer un nouvel acteur
- la recherche des acteurs dans la saisie des experts fonctionne
- il est possible de créer une nouvelle décision juridique
- il est possible de visualiser les décisions juridiques rattachées à un conflit
- en modification de périmètre, le niveau 1 est maintenant visible
- les informations de localisation ont été rajoutées
- il est possible de saisir de nouveaux articles

Évolutions :
- il est possible de télécharger des pièces jointes (articles, entretiens, décisions juridiques)
- dans la saisie des interventions, le changement de la position d'usage d'activité entraîne le changement du contenu de la liste des activités d'usage
- dans la saisie des périmètres, les objets et les biens supports sont gérés différemment selon le type de périmètre
- les paramètres "périmètre" ont été renommés en paramètres "conflit"
- le menu "autres" a été réorganisé et modifié
- une icône a été rajoutée
