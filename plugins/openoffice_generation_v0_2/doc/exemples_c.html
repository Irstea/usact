Exemples C

Colonnes : chiffres ou lettres ?

Peut-être que ça peut paraître un détail, mais parfois, c'est relativement embêtant de savoir quel est l'index numérique de la colonne BG, par exemple. Il y a un moyen qui permet de récupérer cet index, et vice-versa (trouver la lettre en fonction de l'index).

Il faut mixer les fonctions de tableau PHP avec une méthode de la classe Fonction. Cette classe est automatiquement disponible, pour autant que la classe principale soit chargée (voir point 1 ci-dessous).

<?php
// 1.
require_once ('calc/classes/OpenOfficeSpreadsheet.class.php');
$calc = new OpenOfficeSpreadsheet('exemple_c.ods');

$feuille = $calc->addSheet('Une feuille');

// 2. On récupère l'index numérique de la colonne C et AA
$lettres = Fonction::getLetters(true);

$index_C = array_search('c', $lettres); // colonne 3
$index_AA = array_search('aa', $lettres); // colonne 27

// 3. On peut maintenant récupérer la cellule AA, ligne 1
$cell = $feuille->getCell($index_AA, 1);
?>

Les lettres passées en premier paramètre de array_search doivent être en minuscules, sinon, ça marche pas. A noter également le true passé à la fonction getLetters. On peut mettre true ou false. En mettant true, la lettre "a" correspondra à l'index 1 ("b" sera 2, etc). Avec false, la lettre "a" aura l'index 0 ("b" sera 1, et ainsi de suite). En fait, on décale tout de un en mettant true.

<?php
$lettres = Fonction::getLetters(true);
$string_AA = $lettres[27];
?>

Le bout de code ci-dessus illustre comment récupérer le nom de la colonne en partant de son index numérique, si on le connait, bien évidemment.

Modifier un groupe de cellules

Il est possible de faire une même action sur plusieurs cellules en même temps. Suivant comment, on s'économise pas mal de code. C'est toujours bon à prendre.

<?php
// 1.
require_once ('calc/classes/OpenOfficeSpreadsheet.class.php');
$calc = new OpenOfficeSpreadsheet('exemple_c.ods');

$feuille = $calc->addSheet('Une feuille');

// 2. Cellule du coin haut gauche du groupe
$cell_h = $feuille->getCell(1, 1); // A1

// 3. Cellule du coin bas droite du groupe
$cell_b = $feuille->getCell(7, 10); // G10

// 4. On agit sur toutes les cellules
$feuille->setCellContent('Youpie', $cell_h, $cell_b);
$feuille->setCellBackgroundColor('#FFCCCC', $cell_h, $cell_b);
$feuille->setCellColor('#0000FF', $cell_h, $cell_b);
$feuille->setCellFontWeight('bold', $cell_h, $cell_b);
$feuille->setCellBorder('0.02cm solid #FF0000', $cell_h, $cell_b);
$feuille->setCellWidth('3cm', $cell_h, $cell_b);
$feuille->setCellHeight('3.5cm', $cell_h, $cell_b);

// 5. On envoie le fichier en téléchargement
$calc->output();
?>

En gros, on récupère la cellule haut-gauche (point 2) et bas-droite (point 3) du groupe de cellules. Ensuite, en 4, on agit au niveau de la feuille (et non au niveau de la cellule). Là, en l'occurence, on y met du texte, des couleurs, du gras, des bordures de tous les cotés, des largeurs et des hauteurs. La totale, quoi.

Seulement le tour avec setCellBorderAround()

Le groupement de cellules permet de gérer facilement les bordures qui feraient seulement le tour du groupe de cellules, et non pas le tour de toutes les cellules du groupe (elle est pas clair, ma nuance ?!)

<?php
// 1.
require_once ('calc/classes/OpenOfficeSpreadsheet.class.php');
$calc = new OpenOfficeSpreadsheet('exemple_c.ods');

$feuille = $calc->addSheet('Une feuille');

// 2. Cellule du coin haut gauche du groupe
$cell_h = $feuille->getCell(1, 1); // A1

// 3. Cellule du coin bas droite du groupe
$cell_b = $feuille->getCell(7, 10); // G10

// 4. On agit sur toutes les cellules
$feuille->setCellBorderAround('0.2cm solid #FF0000', $cell_h, $cell_b);

// 5. On envoie le fichier en téléchargement
$calc->output();
?>
